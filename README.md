# Semantic Search Project

This project implements an AI-powered semantic document search system.

## Features
- Document upload (PDF)
- Text extraction (Apache Tika)
- Text chunking
- Embedding generation (Sentence Transformers)
- Java + Python integration

## Tech Stack
- Java (Spring Boot)
- Python (Sentence Transformers)
- Apache Tika

## How it works
Upload → Extract → Chunk → Embed → Search

## Future Improvements
- Vector DB (FAISS)
- Semantic search API
- RAG integration
